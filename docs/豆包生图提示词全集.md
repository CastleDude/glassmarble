## 一、统一风格前缀（公共锚点）

所有提示词共用的底模风格描述：

> **八九十年代中国小学课本插图风格，水彩淡彩手绘，丰子恺画风，温暖怀旧，降饱和度15%，整体偏暖黄调，画面留白干净，边缘微微晕开不勾硬线，像泛黄的老课本纸张。**

## 二、珠珠皮肤（12 张）

每颗珠珠是**正方形特写**，居中一颗玻璃珠，浅色背景或留白。

### 🔵 经典蓝珠

```
一颗蓝色透明玻璃弹珠特写，居中放置，画面极简留白。

右上侧有一块锐利的白色高光斑，
```

## 三、环境场景（8 张）

场景是**宽幅横构图**（16:9 或 9:16 竖屏），第一人称低视角。

### 🟢 外婆的后院

```
八九十年代中国乡村后院，第一人称低视角（从蹲在地上玩弹珠的小孩视角看出去），
前景是一片翠绿偏灰的草地，草地上有几簇随机的小草叶和水彩笔触感，
中景有一道简易的木栅栏，栅栏上搭着晾衣绳，一件白衬衫在微风里轻轻飘，
远景是灰蓝色的天空，天际线处有几棵模糊的树影，
阳光从左上方洒下来，温暖的午后光线（下午三四点），
地面上有一两个小小的浅色土坑，
整体色彩饱和度偏低，温暖怀旧。
水彩淡彩手绘风格，丰子恺画风，笔触感强，边缘柔和不硬线。
画面留白舒适，像八九十年代小学语文课本的插图。
```

### 风格一致性保障

1. 所有提示词末尾统一加上风格锚点
2. 首批生成后挑一张最满意的作为「风格基准图」
3. 后续用豆包「以图生图」功能参考基准图保持一致性
4. 每类元素内部保持一致，各类之间允许微调差异

### 第 4 关 · 河边的卵石滩（5 张）

**① 河滩全景**

```
小河南岸的卵石滩，大大小小的圆石头铺满河岸，被流水打磨得光滑圆润。河水只到脚踝，清澈见底。远处青山如黛，河道蜿蜒隐入山间。夕阳把石滩和水面都染成金色，波光粼粼。几个小孩的身影在石滩上弯腰捡石头。画面开阔而美丽。
```

**② 石头围坑**

```
俯视特写：一片卵石滩上，一个用石头围成的圆圈，像一个小小的竞技场。圆圈中心是一个用手指挖出的小坑。坑边放着几颗彩色弹珠。石头有大有小，颜色各异，但都被流水磨去了棱角。画面构图像一座微缩的古代竞技场。
```

**③ 追弹珠**

```
一个小男孩沿着河岸狂奔，身体前倾，手臂伸向河面。河水中一颗蓝色弹珠正顺流而下，在水面上若隐若现。男孩的脚溅起水花，脸上是焦急的表情。阳光从侧面打过来，勾勒出奔跑的轮廓。画面充满动感和紧迫感。
```

**④ 河边的眼泪**

```
男孩坐在石滩上抱膝哭泣，眼泪挂在脸上。手里空空如也。面前是流淌的河水，弹珠已经消失在视野尽头。周围是漫天的金色夕阳，温暖的光线和男孩的眼泪形成反差。画面色调暖黄但情绪忧伤。
```

**⑤ 被磨圆的石头**


### 第 6 关 · 夏日傍晚的晒谷场（5 张）

**① 空出来的晒谷场**

**② 脚底的温度**


**③ 金色竞技场**

```
几个小孩趴在地上玩弹珠，影子被夕阳拉得老长老长。弹珠在平整的水泥地上滚得又直又远，地面上用粉笔画了线和圈。孩子们的表情认真得像在参加奥运会。整个画面被金色笼罩——金色的地面、金色的弹珠、金色的笑脸。
```

**④ 停车场**

```
同一片水泥地，但现在是夜晚的停车场，停着几辆汽车。路灯发出惨白的光。地面还是那片地面，但周围的环境全变了。一个成年人的影子孤零零地站在空地上，手里拿着一颗弹珠。画面色调偏冷灰，与前面金色的晒谷场形成强烈对比。
```

**⑤ 被收走的乐园**

**⑤ 嗒的一声**

```
男人蹲在荒院当年的位置又挖了一个坑。小男孩开心地蹲在旁边，把一颗蓝色弹珠滚了进去。弹珠落入坑中，激起细小的尘土。夕阳把两人的影子拉得很长很长。画面温暖、圆满、充满传承的意味。"嗒"的一声，和三十年前一模一样。世界是一个圆。
```
